## 🎨 功能模块详解

### 1️⃣ 登录模块 (`Login.vue`)

#### 功能描述
用户身份认证入口，提供安全便捷的登录体验。

#### 主要特性
- ✅ **美观界面**: 渐变背景设计，现代化视觉效果
- ✅ **表单验证**: 实时输入验证，错误提示
- ✅ **响应式设计**: 适配各种屏幕尺寸
- ✅ **用户体验**: 密码显示/隐藏，自动聚焦

#### 界面元素
- **系统标题**: 🏗️ 基坑安全管理系统
- **用户名输入**: 带用户图标的输入框
- **密码输入**: 带锁图标的密码框，支持显示切换
- **登录按钮**: 全宽按钮，点击触发验证
- **默认账号提示**: admin/123456

#### 交互流程
1. 用户输入用户名和密码
2. 系统进行表单验证
3. 验证成功跳转到首页
4. 验证失败显示错误提示

#### 技术实现
```vue
<script setup>
import { ref, reactive } from 'vue'
import { useRouter } from 'vue-router'
import { ElMessage } from 'element-plus'

// 表单数据
const loginForm = reactive({
  username: '',
  password: ''
})

// 表单验证规则
const rules = {
  username: [{ required: true, message: '请输入用户名', trigger: 'blur' }],
  password: [
    { required: true, message: '请输入密码', trigger: 'blur' },
    { min: 6, message: '密码长度不能少于6位', trigger: 'blur' }
  ]
}
</script>
```

### 4️⃣ 设备管理 (`DeviceList.vue`)

#### 功能描述
设备信息管理中心，提供设备的全生命周期管理功能。

#### 主要功能

**搜索筛选**
- 🔍 **设备名称**: 模糊搜索设备
- 📊 **设备状态**: 按状态筛选（正常/故障/维护/报废）
- 🔄 **重置功能**: 清空筛选条件

**数据表格**
- 📋 **设备编号**: 唯一标识
- 📝 **设备名称**: 显示设备类型和编号
- 🏷️ **设备类型**: 测斜仪/水位计/土压力计
- 📍 **安装位置**: 具体安装区域
- 📅 **安装日期**: 设备启用时间
- 🎯 **设备状态**: 彩色标签显示
- 🔧 **上次维护**: 最近维护记录

**操作功能**
- ➕ **新增设备**: 添加新设备记录
- ✏️ **编辑设备**: 修改设备信息
- 🔄 **状态切换**: 正常/故障状态互转
- 🗑️ **删除设备**: 移除设备记录

#### 数据结构
```javascript
// 设备数据示例
{
  deviceId: 'DEV001',
  deviceName: '测斜仪-01号',
  deviceType: '测斜仪',
  location: 'A区-北侧基坑',
  installDate: '2024-01-15',
  status: 'normal',
  lastMaintenance: '2024-03-10'
```

#### 状态管理
- 🟢 **正常**: 绿色标签，设备运行良好
- 🔴 **故障**: 红色标签，设备需要维修
- 🟡 **维护中**: 黄色标签，设备正在维护
- 🔵 **已报废**: 蓝色标签，设备已停用

#### 交互特性
- 📄 **分页功能**: 支持页面大小调整
- ⚡ **实时搜索**: 输入即搜索
- 🔔 **操作确认**: 删除和状态变更需确认
- 📱 **响应式表格**: 移动端适配

### 5️⃣ 工单调度 (`WorkOrder.vue`)

#### 功能描述
工单流程管理中心，实现从报修到验收的完整业务闭环。

#### 工单状态流转
```
待派单 → 已派单 → 进行中 → 已完成 → 已验收
   ↓        ↓        ↓        ↓        ↓
 创建    指派维修   开始维修   完成维修   确认验收
```

#### 主要功能

**筛选查询**
- 📊 **工单状态**: 按流程状态筛选
- ⚡ **优先级**: 按紧急程度筛选
- 🔄 **重置条件**: 清空筛选

**工单列表**
- 🆔 **工单编号**: 唯一工单ID
- 📱 **设备名称**: 关联的设备信息
- 🔧 **故障类型**: 故障分类描述
- 👤 **报修人**: 提交工单人员
- ⏰ **报修时间**: 工单创建时间
- 🎯 **优先级**: 紧急/高/中/低
- 📊 **状态**: 当前流程状态
- 👨‍🔧 **维修人员**: 指派的维修师傅

**操作按钮**
- 👁️ **查看**: 查看工单详情
- 📤 **派单**: 指派维修人员（待派单状态）
- ✅ **验收**: 确认维修完成（已完成状态）

#### 数据结构
```javascript
// 工单数据示例
{
  workOrderId: 'WO001',
  deviceName: '测斜仪-01号',
  faultType: '数据异常',
  reporter: '张工',
  reportTime: '2024-05-06 09:30',
  priority: 'urgent',
  status: 'pending',
  assignee: ''
}
```

#### 优先级分类
- 🔴 **紧急**: 需要立即处理
- 🟠 **高**: 优先处理
- 🟡 **中**: 正常处理
- 🔵 **低**: 可延后处理

#### 业务流程
1. **报修**: 用户发现设备故障，提交工单
2. **派单**: 管理员指派合适的维修人员
3. **维修**: 维修人员现场处理故障
4. **完成**: 维修人员报告处理结果
5. **验收**: 管理员确认维修质量

## 🚀 使用指南

### 环境要求
- **Node.js**: 16.x 或以上版本

## 🐛 问题解决

### 常见问题
1. **端口冲突**: 修改 vite.config.js 中的端口配置
2. **依赖安装失败**: 删除 node_modules 重新安装
3. **路由跳转异常**: 检查路由配置和组件导入
4. **样式显示异常**: 确认 Element Plus 样式导入

### 调试技巧
- 使用 Vue DevTools 扩展
- 查看浏览器控制台错误
- 检查网络请求状态
- 验证路由匹配规则
